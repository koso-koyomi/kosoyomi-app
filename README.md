# 古層診断｜あなたの名と、月のリズム（無料Web診断アプリ）

Xアカウント「古層の暦」のキラーコンテンツ（設計正本＝`..\アカウント設計書.md` §4 目玉1）。
静的HTML+JSのみ。サーバー・AI・外部通信なし。**入力（かな氏名＋生年月日）は端末内ハッシュで処理され、送信も保存もされない。**

## 仕組み

- かな氏名（正規化: 半角カナ→全角→ひらがな統一・空白除去）＋生年月日 → FNV-1aハッシュ×2系統 → 主古層(12)×裏古層(12)＝**144通り**。同じ入力なら必ず同じ結果。
- 月相は基準新月(2000-01-06 18:14 UTC)からの朔望月計算。新月・満月の夜だけ「作法」ブロックが追加表示される（再点火装置）。
- 希少度の偏り演出はしない（設計書§9で却下済み）。判定は12層一様分布（実測±0.5%）。

## 公開手順（GitHub Pages）

1. このフォルダをリポジトリにして GitHub Pages を有効化（`.nojekyll` 同梱済み）
2. 公開URLが決まったら:
   - `js/config.js` の `siteUrl` を実URL（末尾`/`）に変更、`xUrl` を実ハンドルに変更
   - `python tools/gen_share_pages.py --site-url https://実URL/` を実行（share12枚とindexのog:imageを書き換え）
3. X固定ポストとプロフィールのリンク欄に診断URLを設定

※OGP画像のURLは絶対URL必須のため、手順2を忘れるとXでカードが出ない（プレースホルダ `koso-koyomi.example.com` のまま）。

## tools

- `gen_share_pages.py` — シェアページ12枚を生成/再生成（`--site-url`）
- `render_og.py` — OGP画像13枚をヘッドレスChromeで書き出し（紋章SVGの実装をそのまま撮影するので表示と完全一致）。紋章を変えたら再実行
- `test_logic.js` — `node tools/test_logic.js`。決定性（表記ゆれ正規化込み）・144通り全出現・分布一様性・月齢検算
- `qa_result.html` — `/tools/qa_result.html` をブラウザで開くとサンプル入力で結果画面を自動表示（目視QA用）
